A run is a single execution of a [task](/help-center/onboard/scouts/tasks). Use a run when you need one specific report, document, or page.

<Frame>
  <img src="/images/%23run.png" alt="Run interface" />
</Frame>

<Info>
  The customer must first authenticate the source via their magic link. Without that, the Scout has no credentials to sign in with.
</Info>

## Start a run

1. Open the Scouts section and select the task.
2. Click **Run** and provide any inputs the task needs (search term, report name, etc.).
3. Watch it execute live — the Scout shows you each step as it goes.

When the run finishes, the result is available on the run's page. You can also feed it directly into a workbook sheet.

## Status

Every run is tracked. You'll see when it started, when it finished, what it returned, and any errors. Past runs stay accessible on the `/tasks` page.
